### 1. Quick start

```sh
// get the repository
$ cd nodeApp/
$ npm install
$ bower install

$ npm start
//or:
$ node bin/www
```
Then go to browser:
* <code>http://localhost:3000/webmd</code>
or:
* <code>http://localhost:3000</code>
to start!
